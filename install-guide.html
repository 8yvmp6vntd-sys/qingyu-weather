添加晴雨天气到主屏幕

请用 iPhone Safari 或安卓 Chrome/Edge 打开本页面。添加后主屏幕会出现 App 图标，之后可像 App 一样打开。

iPhone Safari：

  1. 点击 Safari 底部的分享按钮。
  2. 向下滑动，选择“添加到主屏幕”。
  3. 确认名称为“晴雨天气”。
  4. 点击右上角“添加”。

安卓 Chrome/Edge：

  1. 点击浏览器右上角的菜单按钮。
  2. 选择“添加到主屏幕”或“安装应用”。
  3. 确认名称为“晴雨天气”。
  4. 点击“添加”或“安装”。
打开晴雨天气

如果没有看到安装入口，请确认使用的是 Safari、Chrome 或 Edge，而不是微信、QQ 或其他内置浏览器。
